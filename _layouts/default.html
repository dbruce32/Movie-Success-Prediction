About the Project

This project explores how various factors contribute to a movie's success. By analyzing features like budget, revenue, and genres, we aim to build predictive machine learning models that can accurately forecast box office performance in the form of revenue.

Key Features

  • Data preprocessing, including cleaning, transformation, and dimensionality reduction
  • Machine learning models for predicting box office revenue and critical reception
  • Feature analysis and visualization to understand key contributors to movie success
